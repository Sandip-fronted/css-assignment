(1) What are the benefits of using CSS?

ANS =>
1. Saparation of of Content and Presentation: CSS allows you to separate the structure and content of a web page (HTML) from its presentation and styling. This separation makes it easier to maintain and update a website since you can change the design without altering the underlying content.
2. Consistency: CSS enables consistent styling across a website. By defining styles in a single CSS file, you can ensure that fonts, colors, and layout elements are applied uniformly across all pages, which enhances the overall user experience.
3. Improved Accessibility: CSS can be used to create more accessible websites by providing semantic markup for screen readers and assistive technologies. It allows for better control over the visual representation of content while preserving a logical document structure.
4. Faster Loading Times: CSS files can be cached by browsers, which helps reduce page load times for returning visitors. This results in a more efficient and responsive user experience.
5. Responsive Design: CSS facilitates the creation of responsive web designs that adapt to different screen sizes and devices. Media queries in CSS allow you to define specific styles for various viewports, making your site mobile-friendly and ensuring a consistent experience on different devices.
6. Efficiency and Maintainability: CSS enables you to make global design changes by modifying a single style definition. This reduces redundancy and makes your code more efficient and easier to maintain. It's especially useful for large websites and web applications.
7. Search Engine Optimization (SEO): Clean, well-structured HTML and CSS can improve SEO. Search engines favor websites with organized content and proper markup, making it easier for them to index and rank your pages.
8. Print-Friendly Pages: CSS can be used to create printer-friendly versions of web pages, ensuring that the content is properly formatted when users print pages from your website.
9. Cross-Browser Compatibility: CSS helps achieve consistent visual results across different web browsers. It allows you to address browser-specific issues and implement workarounds when necessary.
10. Styling Flexibility: CSS offers a wide range of styling options, including fonts, colors, gradients, animations, transitions, and more. This flexibility allows you to create visually appealing and engaging web designs.
11. Maintainable Code: CSS promotes a more organized and structured approach to styling, which makes your codebase cleaner and more maintainable, particularly in collaboration with other developers.
12. Integration with Other Technologies: CSS can be easily integrated with other web technologies, such as JavaScript and HTML, to create interactive and dynamic web applications.

(5) What do you understand by CSS opacity?

ANS =>
The CSS property for setting opacity is opacity. Here's how it works:
- A value of 0 makes the element fully transparent, so it is not visible at all.
- A value of 1 makes the element fully opaque, so it is completely visible.
- Values between 0 and 1 create varying degrees of transparency, allowing you to see through the element to the content or background behind it.

(13) What are the ways to integrate CSS as a web page?

ANS =>
- There are three ways to integrate CSS into a web page:
1. Inline CSS: This approach involves adding CSS rules directly to the HTML element using the style attribute.
2. Internal CSS: This approach applies CSS styles to a specific HTML document. Internal CSS is defined inside an HTML document using the style attribute within the head tag of an HTML.
3. External CSS: This approach involves defining the CSS in a separate file and linking it to the HTML document using the link tag.

(14) What is embedded style sheet?

ANS =>
- Embedded Stylesheet: It allows you to define styles for a particular HTML document as a whole in one place. This is done by embedding the (style) tags containing the CSS properties in the head of your document. Embedded style sheets are particularly useful for HTML documents that have unique style requirements from the rest of the documents in your project. However, if the styles need to be applied across multiple documents, you should link to an external style sheet instead of using individual embedded style sheets. Using embedded stylesheets holds a distinct advantage over inline styles which only allow you to address one HTML element at a time.

(16) what are the advantages and the disadvantages of using external stylesheets?

ANS =>
- External stylesheets are used to style multiple HTML pages with a single style sheet. They contain a separate CSS file with a .css extension. The CSS file contains style properties added on selectors (For example class, id, heading, … etc.).

The advantages of using external stylesheets are:
-> Improved maintainability and code organization.
->Enhanced reusability across multiple HTML files.
-> Efficient caching and faster page load times.

The disadvantages of using external stylesheets are:
-> Pages may not render correctly until the external CSS is loaded.
-> Uploading or linking to multiple CSS files may increase your site’s download time, affecting its overall performance.
-> Large-scale projects may face versioning and caching challenges when using external CSS.

(17) What is the meaning of the CSS selector?

ANS =>
-> A CSS selector is a pattern used to select HTML elements that you want to style. It is the first part of a CSS rule and consists of one or more selectors separated by a comma.

Here are some examples of CSS selectors:

-> Element Selector: Selects all elements of a specific type, such as p for paragraphs.
-> Class Selector: Selects all elements with a specific class, such as .example for elements with class=“example”.
-> ID Selector: Selects a single element with a specific ID, such as #example for an element with id=“example”.
-> Attribute Selector: Selects elements with a specific attribute, such as [href] for elements with an href attribute.
-> Pseudo-class Selector: Selects elements based on their state, such as :hover for elements being hovered over.
-> Pseudo-element Selector: Selects parts of an element, such as ::first-line for the first line of text in an element.

(18) What are the media types allowed by CSS?

ANS =>
- CSS allows the following media types :

-> all: Suitable for all devices.
-> print: Intended for printers.
-> screen: Intended for computer screens, tablets, smart-phones, etc.
-> speech: Intended for screen readers that “reads” the page out loud.
-> aural: Intended for speech synthesizers.
-> braille: Intended for braille tactile feedback devices.
-> embossed: Intended for paged braille printers.
-> handheld: Intended for handheld devices (typically small screen, limited bandwidth).
-> projection: Intended for projected presentations, for example projectors.
-> tty: Intended for media using a fixed-pitch character grid, such as teletypes, terminals, or portable devices with limited display capabilities.
-> tv: Intended for television-type devices (low resolution, color, limited-scrollability screens, sound available).
